On saute l'étape2. On le remplace par l'option -a
git commit -a  -m "blabla"   ou   git commit -am "blabla"

Lister les commit ?

git   log 

  Pour quitter le log taper: Q

Aller sur un commit ?

git   checkout  SHADuCommit

Revenir à votre branche
  principale (au commit le plus récent)

git   checkout  master

Renommer un commit ?

git commit   --amend  -m "nouveau message"

Annuler un commit ?
  (après un push)

git   revert  numero

Annuler les 3 derniers
  commits

git   revert  HEAD~3..HEAD

Annuler les changements
  non commités

git   reset  --hard
